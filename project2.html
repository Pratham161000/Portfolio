Single and Group Chat Application

HTMLCSSJavaScriptReact.jsNode.jsMongoDB
Project description

Built a real-time single and group chat application using React.js for the frontend, Node.js for the backend, and MongoDB for data storage. The application allows users to have one-on-one conversations as well as participate in group chats.

Messages are delivered in real time, providing a seamless communication experience. The project demonstrates a full-stack approach to building a modern chat application.
