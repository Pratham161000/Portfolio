Single and Group Chat Application using Full Stack

Project Description: Built a real-time Single and Group chat application using React.js for the frontend, Node.js for the backend, and MongoDB for data storage. The application allows users to have one-on-one conversations as well as participate in group chats. Messages are delivered in real-time, providing a seamless communication experience. The project demonstrates the implementation of a full-stack solution for building a chat application.
