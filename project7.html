Project description

OLP-forStudents (Online Learning Platform for Students) is a course-selling website built on WordPress using Elementor and a set of supporting plugins. It gives students a place to browse and enrol in paid video courses, with secure online payments handled through Razorpay integration.

Each course tracks a student's completion percentage so they always know how far along they are, and includes an instructor-defined quiz that students must pass to complete the course. On the admin side, instructors can register themselves directly on the site and manage their own presence across the header and footer.
