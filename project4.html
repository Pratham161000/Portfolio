Speech-Based Music Player

PythonKotlinNLPMachine Learning
Project description

Developed a speech-based music player application for Android and Python platforms. The application allows users to play music and provides personalized recommendations based on natural language processing (NLP) and machine learning (ML) algorithms.

Users can use voice commands to control playback and interact with the application, which is integrated with online music platforms to fetch and recommend music based on listening habits — making it easy and enjoyable to discover new songs.
