MOOC Course Android Application

KotlinAndroid
Project description

Created a MOOC (Massive Open Online Course) application to help users enhance their programming skills and earn certificates with ease. The application features various courses, including Python and many others, allowing users to learn at their own pace.

The application includes in-video quizzes to ensure maximum learning potential and engagement, with a user-friendly interface and a seamless learning experience for anyone looking to expand their programming knowledge and skills.
